# TinyCraft
A simple Minecraft clone written in C. 
--- 
## Installation 
1. Fork the repo 
```bash 
git clone https://github.com/DareksCoffee/tinycraft.git
```
2. Build TinyCraft 
```bash 
make -j$(nproc)
./tinycraft
```
--- 
## Dependencies 
GLFW 3.x
Glew 2.x
